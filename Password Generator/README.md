# Password Generator (Python)

A simple Password Generator built using Python.
This project was created as part of my learning journey in Python fundamentals.

## 📌 About the Project
The program generates a random password based on:
- Total password length
- Number of letters
- Number of digits
- Number of symbols

It also includes basic input validation and improves user experience by showing remaining characters.

## 🧠 Concepts Used
- Variables
- Input & Output
- Lists
- Conditions (if / else)
- Loops
- Python standard libraries:
  - random
  - string

## ▶️ How It Works
1. The user enters the total password length.
2. The user specifies how many:
   - letters
   - numbers
   - symbols
3. The program validates the input.
4. A randomized password is generated and displayed.
